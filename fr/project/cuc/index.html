Vérificateur d'URL

18 Mar 2023
Code
Photo de andrewtneel (Unsplash)

CUC (prononciation française : [cuc] / λευκός) est un outil CLI très simple pour vérifier un des différents statuts HTTP d’un site web, par exemple si une page web est disponible (200) ou non trouvée (404).

C’est délicieux comme un TUC (cracker), Biscuit de la société française LU, mais avec un C, pour Console.
