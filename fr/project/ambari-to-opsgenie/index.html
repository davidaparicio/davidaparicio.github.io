Ambari vers Opsgenie

29 Apr 2024
Code
Photo de rgaleriacom (Unsplash)

Le projet Go “ambari-to-opsgenie” héberge un serveur conçu pour transférer les alertes Ambari vers OpsGenie. Il inclut des intégrations supplémentaires telles que Blinky et xbar pour MacOS, permettant une personnalisation via des fichiers de configuration chiffrés par SOPS/Age. Ce projet est open source sous licence MIT. Plus d’informations sur le repository Github ici.
